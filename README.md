<div align="center">
  <p>
    📌 Para expandir o elevador de <strong>3 para 6 andares</strong>, foi necessário ampliar o hardware,
    utilizando <strong>6 LEDs</strong> e <strong>6 botões</strong> conectados aos pinos digitais do Arduino.
  </p>

  <p>
    🔄 O sistema passou a utilizar <strong>switch case</strong> combinado com
    <strong>laços de repetição (for)</strong>, permitindo que o elevador percorra cada andar
    individualmente, simulando o deslocamento real entre os níveis do prédio.
  </p>

  <p>
    ⏱️ Durante o movimento, os LEDs são acendidos e apagados sequencialmente com
    <strong>delay()</strong>, criando o efeito visual do elevador subindo ou descendo.
    Ao final de cada trajeto, a variável responsável pelo <strong>andar atual</strong> é atualizada,
    permitindo que o sistema memorize a posição correta para a próxima chamada.
  </p>
</div>
